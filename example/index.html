What does this Plugin do?

This Plugin is designed to make mobile (and smaller screen) browsing easier and less intrusive to valuable screen real estate.

Once activated, sbResponsiveNavigation uses the original HTML markup for desktop and combines it with some CSS media queries and a little jQuery to create a mobile friendly navigation.

There are two navigation types on offer - a slide-down button style for smaller menu's and a Select box dropdown for larger menus with multiple levels. Either can be activated through the plugin options with no additional markup required.
